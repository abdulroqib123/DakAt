# 🌙 DakAt — Dark Mode for Every Website

DakAt is an open‑source browser extension by **Scybud** that brings consistent, elegant dark mode to any website.  
Designed for simplicity, speed, and accessibility — DakAt automatically adjusts colors, contrast, and brightness to reduce eye strain while preserving readability and design integrity.

---

## ✨ Features
- 🌓 **Universal Dark Mode** — Works across most websites automatically.  
- 💾 **Lightweight** — Minimal resource usage, optimized for performance.  
- 🔒 **Privacy‑Respectful** — No tracking, no data collection.

---

## 🧩 Installation (Manual)
DakAt is currently under review for the **Microsoft Edge Add‑ons Store**.  
Until approval, you can install it manually:

1. Clone or download this repository:
   ```bash
   git clone https://github.com/abdulroqib123/DakAt.git
2. Open your browser’s Extensions page.

3. Enable Developer Mode.

4. Click Load unpacked and select the DakAt folder.

The extension will appear in your toolbar — toggle dark mode on any site instantly.
